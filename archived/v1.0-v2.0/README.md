# Version 1.0-2.0 Archive

This directory contains assets and design documents for Pin-Ball versions 1.0 through 2.0.

## Contents

### Assets
- `assets/sprites/` - Sprite images for sports-themed obstacles (basketball hoop, baseball bat, soccer goal, etc.)
- `assets/sounds/` - Sound effects (procedurally generated placeholder sounds)

### Design Documents
- `design/` - Complete design documentation for version 2.0 features including:
  - Monetization systems
  - Mobile platform support
  - Upgrade mechanics
  - Battle pass system
  - Daily challenges

## Version History

### Version 1.0.0
- Maze pipe system (replaced CurvedPipe with TileMap-based maze)
- Bug fixes for ball release and texture UIDs

### Version 0.2.0 (Commercial Optimization)
- Sports-themed obstacles
- Curved ramp system
- Pipe guide system
- Enhanced hold mechanics
- Procedural sound generation

### Version 0.1.1
- Visual ball launcher
- Ball queue system
- Random static obstacles
- Enhanced boundary system

### Version 0.0.1
- Initial release with basic pinball physics
- Flippers, walls, scoring system
